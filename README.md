# Breakout

Simple breakout stlye game.  
Created with Game Maker Studio 2.

Controls:  
--------------  
A/Left = Move Left  
D/Right =Move Right  
W/Up = Move ball a tiny bit up  
S/Down = Move ball at iny bit down  
Mouse Left = Launch ball when in aim mode
